# Project Name: Poker

## Project Info
- Date: November 2025
- Created by: Lavrenty "Larry" Orlov

## Overview
Description: This is a text-based simplified version of a card game called Five-Card Draw Poker, played between a player and a computer. This program's capabilities include handling user input in Command Prompt, logical decision-making, checking for valid user input, passing in values directly or by reference, and random number generation.

## Features
- Controls the complete gameplay of Five-Card Draw Poker, features continuous user interaction with the terminal, tests for numerical edge cases, includes a main menu, displays game rules, and gives the player the option to leave the game.
- Modular design with separate '.c' and '.h' files
- Easy to run in Visual Studio

## Tech Stack
- Language: C
- IDE: Visual Studio Community

## How to Run
1. Clone the repository on your device's command-line interface:
- ```git clone https://github.com/LavrentyOrlov/PokerPA7.git```
2. Open the project in Visual Studio.
3. Build the solution.
4. Run the program.
